# Face-Reconstructor-DL

Using unsupervised deep learning technique or rather self supervised deep learning technique of autoencoders, this program helps to reconstruct facial images. For more accurate results code_space could be increased. Links for the required datasets are given below.

- [Attribute Names](https://www.cs.columbia.edu/CAVE/databases/pubfig/download/lfw_attributes.txt)
- [Image Names](http://vis-www.cs.umass.edu/lfw/lfw-deepfunneled.tgz)
- [Raw Images](http://vis-www.cs.umass.edu/lfw/lfw.tgz)
